START YOUR SITE NOW — Sito + Chatbot AI

CONTENUTO:
- index.html  → Pagina principale del sito
- css/style.css → Stili CSS completi
- js/script.js  → JavaScript completo con chatbot e generatore demo

COME USARE:
1. Estrai tutti i file nella stessa cartella.
2. Apri index.html nel browser.
3. Clicca il pulsante "DEMO GRATUITA" in basso a destra.
4. Scegli "Genera una landing" e completa le 10 domande.
5. Apri l'anteprima oppure richiedi la proposta su WhatsApp.

FUNZIONI DEL CHATBOT:
- Genera landing personalizzate con wizard guidato.
- Salva landing e contatti in localStorage.
- Mostra, apre ed elimina landing salvate.
- Esporta i dati in start-your-site-now-export.json.
- Risponde su servizi, prezzi, funzionamento, SEO, portfolio e contatti.
- Gestisce le domande fuori tema con un invito utile.
- Ricorda nome utente e ultimo argomento in localStorage.
- Include typing indicator, risposte rapide e scroll automatico.
- Espone risposte e keyword in strutture JavaScript facili da estendere.

NOTA:
- Il chatbot usa logica JavaScript locale, senza API esterne.
- Tutti gli stili sono già definiti nel codice.
